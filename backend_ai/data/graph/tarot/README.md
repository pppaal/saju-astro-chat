# Tarot GraphRAG Data

This folder contains GraphRAG-friendly CSVs for tarot knowledge graph nodes and edges.

- `nodes_*.csv`: graph nodes (cards, spreads, suits, combinations)
- `edges_*.csv`: graph edges (card-to-combination links, etc.)

TarotRAG (text RAG) files live under:
`backend_ai/data/graph/rules/tarot`
