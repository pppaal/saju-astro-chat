**목표**: 분기 1-2명 협업

**타겟**:

### 1.2 플랫폼별 전략

#### YouTube

**장점**:
- 깊이 있는 제품 설명 가능
- 높은 전환율 (5-10%)
- 지속적인 트래픽 (에버그린 콘텐츠)

**콘텐츠 형식**:
- "DestinyPal 솔직 리뷰 + 사용법 튜토리얼" (15-20분)
- "내 사주 vs 점성술 비교 분석" (10분)
- "Destiny Match로 운명의 짝 찾기" (8-12분)

**목표 지표**:
- 조회수: 50,000+
- 참여율: > 5%
- 앱 다운로드: 500+

---

#### Instagram

**장점**:
- 시각적 임팩트 강함
- Z세대 타겟 적합
- 스토리 공유 용이

**콘텐츠 형식**:
- 릴스: "30초 안에 내 사주 분석받기" (30-60초)
- 스토리: "오늘의 운세" 시리즈 (15초 × 10개)
- 피드 포스트: 리딩 결과 스크린샷 + 후기

**목표 지표**:
- 릴스 조회수: 100,000+
- 스토리 링크 클릭률: > 10%
- 앱 다운로드: 300+

---

#### TikTok

**장점**:
- 바이럴 가능성 높음
- 젊은 층 도달
- 낮은 제작 비용

**콘텐츠 형식**:
- "나랑 가장 궁합 좋은 띠는?" 챌린지
- "AI가 분석한 내 전생" 시리즈
- "친구랑 사주 궁합 테스트" 듀엣 영상

**목표 지표**:
- 영상 조회수: 500,000+
- 해시태그 참여: 5,000+
- 앱 다운로드: 1,000+

**개요**: 인플루언서 전용 코드로 전환율 추적

**구조**:

- 혜택: Pro 플랜 첫 달 20% 할인

- 기본 협업비: $300-500
- **추가**: 전환당 $2 (매출 쉐어 10%)

**예시**:
```

총 보상: $900
```

---

### 6.2 ROI 계산

**공식**:
```
ROI = (총 매출 - 총 비용) / 총 비용 × 100%
```

**예시 (타로마스터)**:
```
총 비용: $1,000 (협업비)
가입: 450명
유료 전환: 32명
평균 LTV: $86

총 매출: 32 × $86 = $2,752
순이익: $2,752 - $1,000 = $1,752
ROI = $1,752 / $1,000 = 175% (6.4x)
```

- [ ] 평가 기준 적용 (우선순위 선정)
- [ ] Outreach 템플릿 준비
- [ ] UTM 트래킹 시스템 구축
- [ ] 프로모션 코드 생성 도구 준비
- [ ] 계약서 템플릿 법률 검토

### Launch (협업 진행)

- [ ] 초기 컨택 (주 10명)
- [ ] 응답률 모니터링 (목표: 30%)
- [ ] 협의 진행 (가격, 일정)
- [ ] 계약서 체결
- [ ] 제품 제공 (Pro 플랜)
- [ ] 콘텐츠 가이드라인 공유

### Post-Launch (협업 후)

- [ ] 성과 추적 (일일/주간)
- [ ] 인플루언서 피드백 수집
- [ ] 우수 파트너 재협업
- [ ] 월간 성과 리포트 작성
- [ ] ROI 분석 및 최적화
