### 슬라이드 2: 문제 정의

**목적**: 투자자가 공감할 수 있는 문제 제시

**스토리**:
> "전세계 1.4억 명이 점술 앱을 사용하지만, 동양과 서양 점술을 따로 봐야 하고, 대부분 규칙 기반으로 정확도가 낮습니다."

**구성**:
```
문제 1: 파편화
- 사주는 한국 앱, 점성술은 미국 앱
- 통합 분석 불가능

문제 2: 낮은 정확도
- 규칙 기반 알고리즘 (30년 전 기술)
- 개인화 부족

문제 3: 신뢰도 부족
- 오프라인 점집은 비싸고 부담스러움
- 온라인은 재미 위주, 깊이 없음
```

**데이터 포인트**:
- 글로벌 점술 시장: $12B
- 한국 사주 앱 MAU: 2M+
- Co-Star MAU: 5M+

---

### 슬라이드 3: 솔루션

**목적**: DestinyPal의 독보적 가치 제시

**핵심 메시지**:
> "AI가 동서양 점술을 융합하여 개인 맞춤형 운세를 제공하는 세계 유일의 플랫폼"

**차별화 3가지**:

**1. 융합 시스템 (세계 최초)**
```
사주명리 (동양) + 점성술 (서양) + 타로 (보편)
→ 통합 AI 분석 → 종합 리딩
```

**2. 기술 깊이**
- 1,450+ 라인 독자 알고리즘
- GPT-4o 기반 AI 분석
- PersonaMemory 시스템 (누적 학습)

**3. 네트워크 효과**
- Destiny Match: 운명의 짝 찾기
- 궁합 분석으로 바이럴 유도

---

### 슬라이드 4: 시장 기회 (TAM/SAM/SOM)

**구조**:
```
TAM (Total Addressable Market)
│  글로벌 점술 시장: $12B
│  글로벌 앱 사용자: 1.4억 명
│
└─ SAM (Serviceable Addressable Market)
   │  아시아 + 영어권: $3B
   │  타겟 사용자: 5,000만 명
   │
   └─ SOM (Serviceable Obtainable Market)
      │  Year 1: $70M (1M MAU)
      │  Year 3: $350M (5M MAU)
      │  Year 5: $1B (10M MAU)
```

**시장 성장률**:
- CAGR: 15% (2025-2030)
- Z세대 점술 앱 사용: +300% (최근 3년)

---

### 슬라이드 5: 제품 데모

**목적**: 제품을 실제로 보여주기

**데모 시나리오**:

**1. 온보딩 (30초)**
```
1. 소셜 로그인
2. 생년월일 입력
3. 즉시 첫 리딩 제공
```

**2. 핵심 기능 (1분)**
```
- 사주 분석: "내 사주 팔자는?"
- 점성술: "2026년 운세"
- 타로: "오늘의 카드"
- Destiny Match: "운명의 짝 찾기"
```

**3. AI 대화 (30초)**
```
사용자: "올해 이직해도 될까요?"
AI: [개인 사주 + 대운 + 현재 행성 위치 분석]
   → 맞춤형 조언 제공
```

**비주얼**:
- 스크린샷 4-5개
- 또는 2분 데모 영상 (녹화)

## 6. 프레젠테이션 팁

### 6.1 Do's

- ✅ **열정적으로**: 에너지가 투자자에게 전달됨
- ✅ **간결하게**: 슬라이드당 1-2분
