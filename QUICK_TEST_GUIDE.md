# ⚡ 빠른 테스트 가이드

공유 기능을 5분 안에 테스트하는 방법입니다!

## 🚀 시작하기

### 1단계: 서버 시작 (1분)
```bash
npm run dev
```
✅ 서버가 http://localhost:3000 에서 실행됩니다.

---

## 🎯 빠른 테스트 시나리오

### 시나리오 A: 궁합 분석 공유 (2분)

**📍 URL**: http://localhost:3000/compatibility

1. **정보 입력** (1분):
   ```
   Person 1:
   - 이름: Alice
   - 날짜: 1995-05-15
   - 시간: 14:30
   - 도시: Seoul (Seoul, KR 선택)

   Person 2:
   - 이름: Bob
   - 날짜: 1993-08-20
   - 시간: 09:15
   - 도시: Seoul (Seoul, KR 선택)
   - 관계: Lover 💕
   ```

2. **분석 실행**:
   - "Analyze Compatibility" 버튼 클릭
   - 결과 로딩 대기 (~10초)

3. **공유 테스트**:
   - 페이지 하단 스크롤
   - "Share Result" 버튼 찾기
   - 버튼 클릭!
   - ✅ 이미지 다운로드됨: `compatibility-result.png`

4. **이미지 확인**:
   - 다운로드 폴더에서 이미지 열기
   - ✅ Alice & Bob 이름 확인
   - ✅ 💕 이모지 확인
   - ✅ 궁합 점수 원형 차트 확인
   - ✅ 우주 배경 확인

---

### 시나리오 B: 데스티니 매치 공유 (2분)

**⚠️ 사전 준비**:
- 로그인 필요
- 매치 프로필 설정 필요
- 최소 1개 매치 필요

**📍 URL**: http://localhost:3000/destiny-match/matches

1. **페이지 접속**:
   - 로그인하지 않았다면 로그인
   - 프로필이 없다면 `/destiny-match/setup`에서 생성

2. **매치 확인**:
   - 매치 목록 확인
   - 각 매치의 궁합 점수 확인

3. **공유 테스트**:
   - 상단의 "🌟 공유하기" 버튼 찾기
   - 버튼 클릭!
   - ✅ 이미지 다운로드됨: `destiny-match-result.png`

4. **이미지 확인**:
   - ✅ "🌟 Destiny Match" 제목 확인
   - ✅ 내 이름 확인
   - ✅ 매치 개수 확인 (큰 숫자)
   - ✅ 최고 매치 이름 & 점수 확인
   - ✅ 원형 차트 확인

## 🐛 문제가 있나요?

### 공유 버튼이 안 보여요
1. 페이지 새로고침 (Ctrl+R / Cmd+R)
2. 브라우저 콘솔 확인 (F12)
3. 서버 재시작

### 이미지가 다운로드 안 돼요
1. 브라우저 팝업 차단 확인
2. 다운로드 폴더 권한 확인
3. 다른 브라우저 시도

### 이미지가 비어있어요
1. 브라우저 콘솔에서 오류 확인
2. Canvas API 지원 확인 (최신 브라우저 사용)
3. 개발자 도구에서 네트워크 탭 확인

## 📱 모바일에서도 테스트하기

1. **같은 네트워크에 연결**
2. **내 IP 주소 확인**:
   ```bash
   # Windows
   ipconfig

   # macOS/Linux
   ifconfig
   ```
3. **모바일에서 접속**:
   ```
   http://[내IP주소]:3000
   ```
4. **공유 버튼 클릭**:
   - iOS: 네이티브 공유 시트 열림
   - Android: 네이티브 공유 시트 열림
