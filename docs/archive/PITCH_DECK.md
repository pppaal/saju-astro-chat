# DestinyPal - 투자자 피치 덱

## 한 문장 요약

**"AI가 해석하는 동양 사주 + 서양 점성술 + 타로 + 융 심리학 기반 운세 & 매칭 플랫폼"**

---

## 문제 (Problem)

### 1. 운세 시장의 낙후된 UX
- 기존 운세 앱: 복사/붙여넣기 수준의 뻔한 문장
- 개인화 없음 - 같은 띠면 전부 같은 운세
- 신뢰도 낮음 → 유저 이탈

### 2. 점술사 의존 모델의 한계
- 비용: 1회 상담 5~30만원
- 접근성: 예약, 대면, 시간 제약
- 품질: 점술사 역량에 따라 천차만별

### 3. 데이팅앱의 피상적 매칭
- 외모 중심 스와이프
- 깊은 궁합/성격 분석 부재
- 매칭 후 대화 단절 문제

---

## 솔루션 (Solution)

### DestinyPal = AI 점술사 + 심리상담사 + 궁합 매칭

| 기능 | 설명 |
|------|------|
| **사주팔자** | 생년월일시 기반 사주 분석, 일주론, 오행 밸런스, 대운/세운 |
| **서양 점성술** | 태양/달/상승궁 + 11가지 고급 기법 (Solar Return, 프로그레션, 하모닉 등) |
| **AI 타로** | 실시간 카드 해석, 융 심리학 프레임워크, 커플 공유 타로 |
| **꿈 해석** | 한국 전통 해몽 + 융 원형 분석 + 스토아 철학 |
| **수비학** | 이름/생년월일 기반 숫자 운명 분석 |
| **주역 (I-Ching)** | 64괘 기반 심층 분석 |
| **Destiny Match** | 사주/별자리 궁합 기반 틴더 스타일 데이팅 |
| **AI 상담사** | 5단계 심리상담 프로세스, 위기감지 시스템 |
| **운명 캘린더** | 날짜별 운세 등급 + 카테고리별 분석 (재물/연애/건강 등) |

## 부록 C: AI 엔진 상세

### RAG (검색증강생성) 시스템

우리 AI가 똑똑한 이유: **자체 지식 베이스**

```
사용자 질문
    ↓
┌─────────────────────────────────────┐
│  1. 질문 분석 (의도 파악)            │
├─────────────────────────────────────┤
│  2. 지식 검색 (우리 코퍼스)          │
│     - 429개 꿈 해몽 DB               │
│     - 137개 융 심리학 자료           │
│     - 78개 타로 해석 패턴            │
│     - 64괘 주역 해석                 │
│     - 사주 이론 자료                 │
├─────────────────────────────────────┤
│  3. GPT-4o에 컨텍스트 주입           │
├─────────────────────────────────────┤
│  4. 개인화된 답변 생성               │
└─────────────────────────────────────┘
    ↓
맞춤형 해석 결과
```

### 5단계 심리상담 엔진

```
1단계: 공감 (Empathy)
   └─ 사용자 감정 인식 및 수용

2단계: 탐색 (Exploration)
   └─ 사주/별자리 기반 성격 분석

3단계: 통찰 (Insight)
```
