## 1. 유닛 이코노믹스

### 1.1 ARPU (Average Revenue Per User)

#### 현재 추정

**프리미엄 플랜 구성**:

```
Free: $0/월
Starter: $4.99/월
Pro: $9.99/월
Premium: $19.99/월
```

**전환 믹스 (현재)**:

```
Free → Starter: 50%
Free → Pro: 35%
Free → Premium: 15%

ARPU = ($4.99 × 0.50) + ($9.99 × 0.35) + ($19.99 × 0.15)
     = $2.50 + $3.50 + $3.00
     = $9.00/월
```

#### 최적화 목표

**전환 믹스 (목표)**:

```
Starter 비율 감소 → Pro/Premium 증가
Free → Starter: 30%
Free → Pro: 45%
Free → Premium: 25%

ARPU = ($4.99 × 0.30) + ($9.99 × 0.45) + ($19.99 × 0.25)
     = $1.50 + $4.50 + $5.00
     = $11.00/월 (+22%)
```

**최적화 전략**:

- Pro 플랜 가치 강조
- Premium 온보딩 시 할인 (첫 달 30% off)
- Starter 플랜 혜택 축소

### 1.2 CAC (Customer Acquisition Cost)

#### 현재 CAC

**마케팅 채널별**:

| 채널                 | 월 지출    | 신규 가입 | CAC       |
| -------------------- | ---------- | --------- | --------- |
| 페이스북/인스타 광고 | $2,000     | 350       | $5.71     |
| 구글/네이버 SEM      | $1,500     | 250       | $6.00     |
| 인플루언서           | $1,500     | 450       | $3.33     |
| 유기적 (Organic)     | $0         | 280       | $0        |
| **총계**             | **$5,000** | **1,330** | **$3.76** |

**가중 평균 CAC**: $3.76

#### CAC 최적화 목표

**바이럴 계수 개선**:

```
현재:
- K-Factor: 0.15
- 유기적 비율: 21%

목표:
- K-Factor: 1.5 (추천 보상 증대)
- 유기적 비율: 40%

CAC (최적화) = $3.76 × (1 - 0.40) = $2.26 ✅
```

---

### 1.3 LTV (Lifetime Value)

#### 현재 LTV

**계산**:

```
ARPU: $9/월
평균 구독 기간: 12개월 (Churn 8.3%/월 기준)
Gross Margin: 80% (SaaS 표준)

LTV = $9 × 12 × 0.80 = $86.40
```

#### LTV 최적화 목표

**개선 방안**:

- ARPU 증가: $9 → $11 (가격 최적화)
- 구독 기간 증가: 12 → 18개월 (리텐션 개선)
- Gross Margin 증가: 80% → 85% (AI 비용 절감)

**최적화 후 LTV**:

```
LTV = $11 × 18 × 0.85 = $168.30 (+95%)
```

---

### 1.4 LTV/CAC Ratio

**현재**: $86.40 / $3.76 = **22.9x** (매우 건강)

**목표**: $168.30 / $2.26 = **74.5x** (유니콘급)

**벤치마크**:

- 건강한 SaaS: 3x 이상
- 훌륭한 SaaS: 5x 이상
- 유니콘 SaaS: 10x 이상
