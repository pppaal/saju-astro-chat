# 프로젝트 유니콘급 평가 보고서

이 프로젝트는 **유니콘급 스타트업 잠재력**을 갖춘 수준으로 평가됩니다. 현재는 **프리-유니콘 단계(Unicorn-in-the-making)**입니다.

#### 프론트엔드

- **최신 스택**: Next.js 16 (App Router), React 19
- **타입 안정성**: TypeScript strict mode (100% 타입 커버리지 목표)
- **UI/UX**: Radix UI, Framer Motion, Chart.js
- **PWA**: 오프라인 지원, 모바일 최적화
- **네이티브 앱**: Capacitor로 iOS/Android 지원

- **프레임워크**: Next.js API Routes

3. **주역 (I Ching)**: 64괘 + 효변 분석

4. **서양점성술 (Western Astrology)**: 26개 파일
   - 트랜싯, 프로그레션, 솔라/루나 리턴
   - 하모닉 분석, 드라코닉 차트
   - 일렉셔널 점성술 (길일 선택)

5. **수비학 (Numerology)**: 생명경로, 운명수, 성격수
