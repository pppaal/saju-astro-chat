## 1. 시장 진입 전략

### 1.1 진입 포지셔닝

**핵심 메시지**:

> "전통 사주명리 + 서양 점성술 + AI가 만나는 세계 유일의 통합 운세 플랫폼"

**차별화 포인트**:

1. 동서양 점술 융합 (세계 최초)
2. AI 기반 실시간 분석
3. Destiny Match (데이팅 기능)
4. 1,450+ 라인 독자 알고리즘

### 1.2 단계별 성장 전략

#### Phase 1: PMF 검증 (0-6개월)

**목표**:

- DAU 10,000+ 달성
- 프리미엄 전환율 3%+
- NPS 50+ 확보
- Destiny Match 일일 활성 매칭 100+

**핵심 액션**:

**1. Destiny Match 바이럴화**

- [ ] 매칭 알고리즘 ML 정교화
- [ ] "Match of the Day" 푸시 알림
- [ ] 매치 성공 스토리 수집 (10쌍)
- [ ] 인스타그램 공유 카드 자동 생성

**2. 온보딩 최적화**

- [ ] Aha Moment를 5분 이내로 단축
- [ ] 튜토리얼 간소화 (3단계로)
- [ ] 첫 리딩 무료 + 즉시 제공
- [ ] A/B 테스트 (5개 변형)

**3. 바이럴 루프 구축**

- [ ] 추천 보상 3 → 5 크레딧
- [ ] 친구 초대 시 양쪽 모두 보상
- [ ] 공유 리딩 OG 이미지 최적화
- [ ] 카카오톡 공유 딥링크

**4. 데이터 수집**

- [ ] Mixpanel/Amplitude 통합
- [ ] 퍼널 분석 자동화
- [ ] 코호트 분석 대시보드
- [ ] 주간 사용자 인터뷰 (10명)

**예상 비용**:

- 인프라: $500/월
- AI 비용: $2,000/월
- 마케팅: $5,000/월 (페이스북, 인스타그램 광고)
- **Total: $7,500/월**

**예상 수익**:

- 10k DAU × 3% 전환 × $8 ARPU = **$2,400/월**
- 적자: -$5,100/월 (투자 필요)

- [ ] 결혼정보회사 API 제공 (듀오, 가연 등)
- [ ] 점집/철학관 SaaS 모델
- [ ] HR 업체 (성격/궁합 테스트)

- AI 비용: $15,000/월

- 인건비: $50,000/월 (10명)
- **Total: $98,000/월**

**예상 수익**:

- 100k MAU × 5% 전환 × $10 ARPU = **$50,000/월 MRR**
- B2B: $50,000/월 (추가)
- **Total: $100,000/월 ($1.2M ARR)**
- 손익분기 달성!

## 2. 타겟 세그먼트

### 2.1 Primary Target: Z세대 여성 (20-29세)

**특징**:

- 점성술/MBTI/사주에 관심
- 인스타그램/TikTok 활성 사용자
- 친구들과 운세 공유 문화
- 연애/진로 고민 많음

**Pain Points**:

- 오프라인 점집은 비싸고 부담스러움
- 기존 운세 앱은 재미 없음
- 동양/서양 점술을 따로 봐야 함

**Value Proposition**:

- AI가 통합 분석 (한 번에 해결)
- 언제 어디서나 즉시 리딩
- 친구들과 궁합 분석 (재미)

### 2.2 Secondary Target: 30대 직장인 (남/여)

**특징**:

- 커리어/결혼 고민
- 프리미엄 서비스 지불 의향 높음
- 정확도와 깊이 중시

**Pain Points**:

- 신뢰할 수 있는 상담사 찾기 어려움
- 시간 부족 (오프라인 방문 불가)

**Value Proposition**:

- 전문가급 AI 분석
- 1:1 프리미엄 상담
- 장기 운세 리포트 (PDF)

## 3. 채널 전략

### 3.1 디지털 마케팅 (70% 예산)

- 타겟팅: 20-35세 여성, 관심사 "점성술", "타로", "MBTI"

- 예산: $1,000/월
- KPI: 영상 조회수 1M+, 공유율 > 5%

#### Google/Naver SEM

- 키워드: "무료 사주", "타로 점", "궁합 보기"
- 예산: $1,500/월
- KPI: CPC < $0.50, 전환율 > 3%

| 카테고리                | 구독자  | 단가       | 월간 협업 수 |
| ----------------------- | ------- | ---------- | ------------ |
