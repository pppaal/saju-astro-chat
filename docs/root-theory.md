# 우리 엔진의 뿌리 이론 (Root Theory)

> 이 캘린더 엔진이 **무엇을 근본 원리로 삼아** 사주·점성을 길흉·타이밍으로 바꾸는지.
> 추측 아님 — 코드에서 추출한 사실(file:line 근거). 3-Opus 코드 분석(2026-05) 기반.
> 증상마다 땜질하지 않으려면 이 뿌리를 기준으로 판단한다.

---

## 0. 한 줄 세계관

> **사주와 점성을 같은 원자(polarity −3~+3 · weight 0~1 · layer · 시간창)로 환원해, 두 전통을 따로 강건집계한 뒤(개수가 아닌 종류가 투표), "그 사람 평소 대비"로 보정하고, 둘이 같은 방향으로 강하게 수렴할 때를 "타이밍"으로 본다. 인생부터 시간까지 같은 연산, 배율만 다르게.**
> = **합성지표(OECD) + 강건통계 + 데이터퓨전을 점술에 적용.**

## 1. 사주 뿌리 — "억부 + 용신" 자평 모델

**핵심 원리**: 운(運)의 **천간 십신**을 그 시기 기운으로 보고, **신강/신약에 따라 십신 길흉이 뒤집히며**, 거기에 **용신 오행 부합**을 더해 −3~+3.

| 요소 | 규칙 | 코드 |
|---|---|---|
| 기운 단위 | 대운/세운/월운/일진의 **천간** 십신 (지지는 별도 축) | `saju-pillar.ts:49-178`, `shared/sibsin.ts:30-45` |
| 스케일 가중 | 대운1.0 / 세운0.85 / 월운0.7 / 일진0.55 | `saju-pillar.ts:71,108,139,174` |
| **길흉(핵심)** | **(A)용신부합** + **(B)신강약×십신×1.5** 합산 | `polarityFromSibsinYongsin` `saju-pillar.ts:242-266` |
| ↳ 신강 | 식상·재성·관성=+(설기·극 균형) / 인성·비겁=−(과다) | `sibsinBase` |
| ↳ 신약 | 부호 반전 (인성·비겁=+ / 식상·재성·관성=−) | 〃 |

## 2. 점성 뿌리 — 트랜짓 어스펙트 + 헬레니즘/현대 혼합

**핵심 원리**: 매일 트랜짓 행성이 본명 점과 맺는 **어스펙트 종류(조화각+/긴장각−) × 관여 행성 길흉(benefic/malefic)**으로 −3~+3.

| 요소 | 규칙 | 코드 |
|---|---|---|
| 기운 단위 | 매일 정오 트랜짓 → 본명(10행성+ASC+MC+카이런+릴리스) 어스펙트 | `astro-transit.ts:59-160`, `transit.ts:131` |
| 각도 | 메이저5(conj/sext/sq/tri/opp) + 마이너5(semisext/quincunx/quintile…) | `astro-transit.ts:16-27` |
| **길흉** | 조화각=길/긴장각=흉 기본 × 행성길흉 증폭 | `inferAspectPolarity` `tagger.ts:80-117` |
| 행성 길흉 | 목성·금성=benefic / 화성·토성·명왕=malefic / 나머지 중립 | `featureMap.ts:204-216` |

## 4. 알려진 갭 / 미배선 (정직)

**🔴 구조 갭 — 반드시 확인**
- **`deriveCellAxes`(두 축·헤드라인·agreement)가 `index.ts` 셀 빌드에서 미호출.** 셀(`cell.derivedScore`)은 아직 옛 `deriveScore`(단일 산술 2단평균)만 씀. 융합 점수층은 **route.ts 경로엔 배선**됐으나 **엔진 셀 자체엔 미배선** → 프로토타입 점수(분포·opposed)와 `cell.derivedScore`가 다를 수 있음. **점수 단일화 미완.**

**🟠 도메인 근사 (단순화)**
- 사주: **지지·지장간을 운 십신서 누락**, **통근/합화/격국 미반영**, **조후 정밀표(`johuYongsin.ts`) 있는데 메인 용신은 월지 계절만**, 강약 5→3단계 압축. 용신(억부)과 십신(억부)을 독립 합산 → **억부 이중계산 위험**.
- 점성: **sect(주야) 계산은 하나 어스펙트 polarity엔 안 씀**(헬레니즘 핵심 누락, Lots/ZR엔 적용=불일치). **applying/separating 계산은 하나 축 점수 미반영**. polarity 규칙이 extractor마다 다름(트랜짓/진행/마이너/행성시).
- 융합: 두 축 raw 음상관(r≈−0.45 일부 차트)은 강도게이트로 *완화*지 *해결* 아님.

**🟢 검증된 것**
- 사주 +쏠림 뿌리(십신 신강약 미반영)는 **수정 완료** — 음수 자연발생, 음상관·분포 동시 정상화.
- 극지 크래시·용신 중복·연애 성별·테마 도배 = 수정 완료.

## 5. 이 문서의 용도

증상(점수 낮다/opposed 높다/도배된다)이 보이면 — **여기 뿌리에서 어느 원리가 흔들렸는지** 먼저 보고, 표면 상수를 땜질하지 말 것. 대부분 증상의 뿌리는 §1~3의 한 원리다(예: 분포 압축·음상관 = §1 사주 +쏠림 한 뿌리였음).
